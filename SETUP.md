# SETUP.md for Minecraft Bedrock Toolkit

## Setup Instructions
1. **Download** the latest version of Minecraft Bedrock Toolkit from the [releases page](https://github.com/Darkentitty/minecraft-bedrock-toolkit/releases).
2. **Extract** the downloaded zip file to your desired installation location.
3. **Install** any prerequisites:
   - .NET Framework 4.7.2 or higher
4. **Run** the `setup.exe` file to initiate the installation process.
5. **Follow** the prompts to complete the installation.

## System Requirements
- **OS**: Windows 10 or later.
- **Processor**: 2 GHz dual-core processor or better.
- **RAM**: At least 4 GB of RAM.
- **Storage**: Minimum of 1 GB of free disk space.
- **Graphics**: DirectX 11 compatible video card.

## Troubleshooting Guide
- **Installation Issues**: If you face issues during installation, make sure you have administrative privileges and that no other setup is running.
- **Application Crashes**: Ensure your system meets the minimum requirements and that no antivirus software is blocking the application.
- **Running the Tool**: If the application does not run, verify that all dependencies are installed correctly.

## FAQ
**Q1: Can I use this toolkit on Windows 7?**  
A1: The toolkit is designed for Windows 10 and later. Windows 7 is not supported.

**Q2: Is there any way to revert changes made by the toolkit?**  
A2: Yes, the toolkit includes an option to restore previous settings in the settings menu.

**Q3: Where can I find support?**  
A3: Support is available through the GitHub Issues page for the project.
